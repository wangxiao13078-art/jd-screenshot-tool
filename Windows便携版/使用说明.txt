========================================
  京东截图编辑工具 - Windows 便携版
  使用说明（无需安装任何软件）
========================================

【第一步】下载 Python 便携版（只需做一次）

1. 打开网址：https://www.python.org/ftp/python/3.11.9/python-3.11.9-embed-amd64.zip
2. 下载这个 zip 文件（约 10MB）
3. 解压到当前文件夹，得到一个 python-3.11.9-embed-amd64 文件夹
4. 把这个文件夹重命名为 python

【第二步】下载 pip

1. 打开网址：https://bootstrap.pypa.io/get-pip.py
2. 右键 → 另存为 → 保存到当前文件夹
3. 双击运行 安装pip.bat

【第三步】安装依赖

1. 双击运行 安装依赖.bat
2. 等待安装完成

【第四步】运行程序

1. 双击运行 启动.bat
2. 程序会打开，开始使用！

========================================
  之后每次使用只需双击 启动.bat
========================================

常见问题：
Q: 提示"Windows 已保护你的电脑"？
A: 点击"更多信息" → "仍要运行"

Q: 程序闪退？
A: 确保按顺序执行了第一步到第三步
